Face Recognition Algorithm Types

Understanding different computational approaches to identify and authenticate human faces in various scenarios and applications.

Eigenfaces
Statistical Method
Principal Component Analysis
PCA-Based Dimensionality Reduction
Fast recognition speed
Lower computational cost
Pioneering technique
Decomposes face images into principal eigenvectors representing key facial variations.
LBPH
Local Feature Method
Local Binary Patterns
Texture Pattern Histogram Analysis
Robust to lighting changes
Good rotation invariance
Real-world reliable
Analyzes local texture patterns by comparing pixel values with neighbors in regions.
Fisherfaces
Discriminant Method
Linear Discriminant Analysis
Maximizes Between-Class Variance
Better discrimination
Handles lighting well
Authentication focused
Enhancement over Eigenfaces using LDA to find linear combinations that separate face classes.
DNN
Deep Learning
Deep Neural Networks
Multi-Layer Neural Architecture
Complex representations
High accuracy
Modern approach
Multi-layer perceptron networks trained with backpropagation on large annotated datasets.
CNN
Convolutional Method
Convolutional Neural Networks
Hierarchical Feature Extraction
Automatic feature learning
99%+ accuracy
State-of-the-art
Convolutional layers detect progressively complex features from edges to facial structures.
HOG
Gradient-Based
Histogram of Gradients
Gradient Orientation Counting
Robust to illumination
Works with SVM
Fast processing
Extracts edge and gradient information by grouping orientations for detection tasks.

Understanding Facial Landmarks

Facial landmarks are precisely localized points on a face that serve as reference markers. These 68 points cover all major facial features.

1-17
Jawline & Contour Defines face boundaries and shape
18-27
Eyebrows Left and right eyebrow positions
28-36
Nose Nasal bridge and tip points
37-48
Eyes Eye contours and outer edges
49-68
Mouth Lip boundaries and contours
